# Runjucks documentation site

Built with [Astro Starlight](https://starlight.astro.build/). The **Node.js API** pages are generated by [TypeDoc](https://typedoc.org/) from the parent [`index.d.ts`](../index.d.ts) into `public/api/`. **Rust API** HTML is generated with `cargo doc` on **`runjucks_core`** into `public/rustdoc/` (served under `/rustdoc/runjucks_core/`).

## Node version

This package follows **Astro’s** current requirement: **Node.js ≥ 22.12** (see [`package.json`](./package.json) `engines`). Use the same major in CI and locally.

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Local dev server |
| `npm run api` | Regenerate TypeDoc only |
| `npm run api:rust` | Regenerate rustdoc into `public/rustdoc/` (requires Rust toolchain) |
| `npm run build` | TypeDoc + rustdoc + static site → `dist/` |

### Optional Bun

From this directory you can run **`bun install`**, **`bun run dev`**, **`bun run build`**, **`bun run api`**, **`bun run api:rust`** instead of npm. CI still uses **Node + npm**; if anything fails under Bun, use Node.

## GitHub Pages

The workflow [`.github/workflows/docs.yml`](../.github/workflows/docs.yml) sets:

- `ASTRO_SITE` — `https://<owner>.github.io/<repo>`
- `ASTRO_BASE_PATH` — `/<repo>/`

For a **user** site (`username.github.io` with no project path), set `ASTRO_BASE_PATH=/` in the workflow or override when building locally.

## Local preview with a base path

```bash
ASTRO_BASE_PATH=/runjucks/ ASTRO_SITE=https://you.github.io/runjucks npm run build
npm run preview
```
